# Step1. Create a npm account

https://www.npmjs.com/

After verifying the email.

# Step 2. Write the code

Initialize a project

```
npm init
```

On the entry point of your project console.log your contact information.

# Step 3. Publish your project

Login to npm from the command line

```
npm login
```

and publish your project

```
npm publish --access public
```

Now, other users can access your information using npx:

```
npx rafael-info
```
